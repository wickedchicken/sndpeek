# sndpeek

### A real-time audio visualization tool (animated, 3D)

**soundlab homepage**:
<http://soundlab.cs.princeton.edu/>

**sndpeek homepage**:
<http://soundlab.cs.princeton.edu/software/sndpeek/>

### Who it is for

- audio/multimedia researchers
- developers, composers
- and performers

### Supported platforms

- MacOS X (CoreAudio)
- Linux (ALSA/OSS/Jack)
- Windows/Cygwin (DirectSound)
- SGI (coming soon)
